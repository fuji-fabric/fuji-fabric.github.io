# Module: chat.history

## Overview
:::module
  This module does:
  
  1. Stores chat messages as history.
  
  2. Delivers them to players when they join the server.


:::
## Color Boxes

:::colorbox-tip

  â—‰ This module works with `Styled Chat` mod.
  
  You can use this module to provide the `chat history` for it.


:::
